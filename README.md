# tmh

Single-binary TUI hub for tmux. Declarative YAML config, live ↔ config sync, dotfile reload, shareable team setup.

## Status

Early development. See [plan](/Users/mark/.claude/plans/robust-imagining-dijkstra.md) for the full specification.

## Install

```sh
# Self-hosted (recommended)
GOPRIVATE=git.mark1708.ru/* go install git.mark1708.ru/me/tmh/cmd/tmh@latest

# Homebrew (direct formula)
brew install https://git.mark1708.ru/me/tmh/raw/branch/main/homebrew/tmh.rb
```
